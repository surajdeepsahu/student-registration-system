# Student Registration System

A comprehensive Student Registration System built with React.js that manages courses, course types, offerings, and student registrations.

## Features

- Course Types Management (CRUD operations)
- Courses Management (CRUD operations)
- Course Offerings Management with filtering
- Student Registration with email validation
- Data persistence using localStorage
- Responsive design with Tailwind CSS
- Toast notifications for user feedback

## Installation

1. Clone the repository:
git clone https://github.com/YOUR_USERNAME/student-registration-system.git
cd student-registration-system
